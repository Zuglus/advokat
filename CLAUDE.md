# advokat — сайт сестры Андрея (адвокат, Санкт-Петербург)

Сайт-визитка адвокатской практики. Территория Андрея (ведёт сам), заказчик — сестра.
Целевой домен в коде — `advokat-sank-peterburg.ru` (константа `baseURL` в `main.go`).

## Состояние прода (проверено 11.07.2026)

**Сайт не опубликован: домен не зарегистрирован** (DNS — NXDOMAIN, whois — «No entries
found», домен свободен). На этом сервере процесса/юнита нет, деплойных конфигов в репо
нет. Публикация — отдельный шаг, которого ещё не было (или домен истёк — тогда
перерегистрировать, пока свободен). Решение за Андреем.

## Стек и запуск

- Go 1.24.13 (закреплён `mise.toml`), stdlib без внешних зависимостей:
  `net/http` + `html/template`.
- Запуск локально: `mise install && PORT=8080 go run .` (порт — из env `PORT`).
- Репозиторий: `github.com/Zuglus/advokat` (публичный GitHub-флоу, PR-ы; миграция
  на Go — PR #31, июнь 2026).

## Структура

- `main.go` — весь сервер одним файлом: конфиг, роутинг, рендер шаблонов, sitemap/SEO.
- `_data/*.json` — **весь контент сайта**: `site.json` (титул/описание), `nav.json`,
  страницы практик (criminal, civil, family, tax, arbitraj, house, legacy, consulting,
  assistance), `contacts*.json`, `about.json`, `usefulLinks.json`, `notFound.json`.
  Правка текста сайта = правка JSON, не шаблонов.
- `templates/` — `base.html` + `pages/` + `partials/`.
- `css/`, `js/`, `images/`, `favicon.ico`, `manifest.json`, `robots.txt` — статика.
- `agreement.docx` — документ соглашения (исходник, отдаётся с сайта).

## Правила

- Контент (тексты практик, контакты) — материя сестры: содержательные правки — только
  по слову Андрея, самодеятельных формулировок «за адвоката» не писать.
- Наружу (деплой, регистрация домена, DNS) — только по явному слову Андрея (правило
  security-шагов из `~/oniwadi/CLAUDE.md` действует).
- Синхронизация — через GitHub (клоны: мак и этот сервер); перед правками `git pull`.
